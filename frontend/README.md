# CoffeeHouse Frontend

The CoffeeHouse frontend is built using **Vite** and **React JSX**, providing a fast and modern development experience.

## Features

- **Vite**: A lightning-fast build tool for modern web projects.
- **React JSX**: Enables building reusable UI components with a declarative syntax.
- **Hot Module Replacement (HMR)**: Instant updates during development.
- **Optimized Build**: Efficient bundling and minification for production.

## Getting Started

### Prerequisites

Ensure you have the following installed:

- [Node.js](https://nodejs.org/) (version 18 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/your-repo/coffeehouse-frontend.git
    cd coffeehouse-frontend
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

### Development

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Build for Production

To create an optimized production build:
```bash
npm run build
```

The build output will be in the `dist` directory.
